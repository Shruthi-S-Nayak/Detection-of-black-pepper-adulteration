Detection of Black Pepper Adulteration
📌 Project Overview
This project focuses on detecting adulteration in black pepper using deep learning and computer vision techniques. It helps identify whether black pepper samples are pure or mixed with impurities.
🚀 Features
* Image-based adulteration detection
* Deep learning models (MobileNet, YOLO)
* Dataset preprocessing and augmentation
* Real-time prediction using a web interface
* User-friendly frontend with templates

## 🛠️ Technologies Used

* Python
* TensorFlow / PyTorch
* OpenCV
* Flask
* HTML, CSS


## 📂 Project Structure

```
├── app.py                 # Main application
├── templates/             # Frontend HTML files
├── requirements.txt       # Dependencies
├── *.ipynb                # Model training notebooks
├── outputs/               # Results
```
⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/Shruthi-S-Nayak/Detection-of-black-pepper-adulteration.git
```

2. Navigate to the project folder:

```
cd Detection-of-black-pepper-adulteration
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
python app.py
```
 Output

* Detects adulterated vs pure pepper
* Displays prediction results visually
